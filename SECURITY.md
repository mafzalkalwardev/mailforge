# Security Policy

## Supported versions

| Version | Supported |
|---------|-----------|
| 1.x     | Yes       |

## Reporting a vulnerability

Please **do not** open public issues for security problems.

Email the maintainer privately or open a GitHub Security Advisory if enabled.

## Notes

- Change `JWT_SECRET` before any public deployment
- Do not commit `.env` or upload real customer email lists to public repos
- SMTP verification connects to third-party mail servers — use responsibly and follow anti-spam laws
